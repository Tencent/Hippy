# 颜色

颜色属性取值具体可以查询[CSS的颜色配置](//developer.mozilla.org/zh-CN/docs/Web/CSS/color_value).

# RGB

Hippy 支持 `rgb()` and `rgba()`。

* `'#f0f'` (#rgb)
* `'#ff00ff'` (#rrggbb)
* `'rgb(255, 0, 255)'`
* `'#f0ff'` (#rgba)
* `'#ff00ff00'` (#rrggbbaa)
* `'rgba(255, 255, 255, 1.0)'`

# HSL

Hippy 也支持 `hsl()` and `hsla()`

* `'hsl(360, 100%, 100%)'`
* `'hsla(360, 100%, 100%, 1.0)'`

# 透明

 `rgba(0, 0, 0, 0)`还有另外一个简写版本:

* `'transparent'`
